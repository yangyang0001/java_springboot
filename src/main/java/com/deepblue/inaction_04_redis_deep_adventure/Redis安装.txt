----------------------------------------------------------------

安装
[root@localhost redis-3.2.8]# make install PREFIX=/home/redis
[root@localhost redis-3.2.8]# cp redis.conf ../redis

启动服务端
    到redis安装目录中启动redis-server
    cd /home/redis/bin
    ./redis-server ../redis.conf

启动客户端
    redis启动客户端的情况:
    ./redis-cli -h 127.0.0.1 -p 6379 -a root
    连接客户端,使用 密码的情况
    ./redis-cli -h 192.168.188.7 -p 6379 -a root
